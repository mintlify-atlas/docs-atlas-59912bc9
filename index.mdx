Jitsi Meet is a fully open-source, WebRTC-powered video conferencing solution built by the Jitsi team at 8x8. Use it directly at [meet.jit.si](https://meet.jit.si), deploy your own instance on any Linux server, or integrate it into your application using the IFrame API or React Native SDK — no proprietary plugins required.

<Steps>
  <Step title="Try Jitsi Meet instantly">
    Visit [meet.jit.si](https://meet.jit.si) and create a room — no account or download required.
  </Step>
  <Step title="Embed in your app">
    Add the [IFrame API](/iframe-api/overview) script to your page and create a `JitsiMeetExternalAPI` instance pointing at any Jitsi server.
  </Step>
  <Step title="Self-host or use JaaS">
    Run your own server with [Debian packages or Docker](/self-hosting), or sign up for [JaaS](/jaas/overview) to skip the infrastructure entirely.
  </Step>
  <Step title="Authenticate and customize">
    Issue [JWT tokens](/configuration/jwt-auth) to control room access, then tune [config.js](/configuration/config-js) to match your product experience.
  </Step>
</Steps>
